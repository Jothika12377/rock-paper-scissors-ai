A real-time Rock Paper Scissors game built using:

- Countdown-based fair gameplay
- Live score tracking
- Computer opponent

## How to Run

1. Install dependencies:
pip install opencv-python mediapipe

2. Download the model file:
hand_landmarker.task

Place it inside:
models/

3. Run:
python rps_camera_tasks.py
